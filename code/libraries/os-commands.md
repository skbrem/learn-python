The most common os commands that I'll use with the `os` library in Python. 

| Command | Function | Use Case |
| --- | --- | --- |
| `os.getcwd()` | **Get** **C**urrent **W**orking **D**irectory. | Finding out where the script is running. |
| `os.listdir()` | List files in a folder. | Auditing a directory for specific logs. |
| `os.mkdir()` | **M**ake **K**ey **D**irectory. | Creating a new "Backup" folder for today's date. |
| `os.rename()` | Rename a file. | Adding a timestamp to an old log file. |
| `os.path.exists()` | Check if a file exists. | Verification before trying to open a config file. |
